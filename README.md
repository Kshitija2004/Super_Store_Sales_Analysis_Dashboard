# Super_Store_Sales_Analysis_Dashboard

This project is an interactive Super Store Sales Dashboard to analyze historical sales data and generate a 15-day sales forecast.

The dashboard helps understand sales performance, profit trends, customer segments, shipping modes, and regional insights, while the forecast assists in short-term business planning.

Objectives:

Analyze overall sales, orders, and profit.

Compare year-over-year performance.

Identify top-performing regions, states, and categories.

Understand customer segments and payment behavior.

Perform 15-day sales forecasting for demand planning.
